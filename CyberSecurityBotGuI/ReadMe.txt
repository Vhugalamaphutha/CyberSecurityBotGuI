Cyber Security Awareness Chatbot(ReadMe)  

Overview

a GUI based cybersecurity chatbot that educates users about online safety. Built with C# Windows Forms.

Features 
-Voice Greeting
-keyword recognition (passwords, phishing, safe browsing, updates, email)
-Random responses from lists
-Follow up handling (“tell me more”, “another tip”)
-Memory (remembers your name and favorite topic)
-Sentiment detection (worried, frustrated, curious)
-Error handling for unknown inputs
-ASCII art display
-Clear chat & play greeting buttons

How to run
1. Open the solution in Visual Studio.
2. Ensure the WAV file is placed in `Resources\Awarness chatbot audio.wav`
3. Press **F5** to build and run.
