 Cybersecurity Awareness Chatbot – Part 3

 Overview
A GUI-based cybersecurity chatbot that educates users about online safety. Built with C# Windows Forms and MySQL.

 Features
-  Voice greeting
-  Keyword recognition (password, phishing, safe browsing)
-  Random responses
-  Follow-up handling
-  Memory (name, favorite topic)
-  Sentiment detection
-  Task Manager with MySQL
-  Quiz Game with 11 questions
-  NLP simulation
-  Activity Log
-  Error handling

Requirements
- .NET Framework 4.7.2+
- MySQL Server
- MySQL Connector/NET

 Setup
1. Install MySQL and create database `cyberbot`.
2. Update connection string in `TaskManager.cs`.
3. Run the application.
